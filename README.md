# **E-Shop Application**

Simple e-commerce project that includes backend and frontend parts where users can create an account, log in and add products/products.

## **Tech Stack**
---
- **Backend**
    - Java 17
    - Spring Boot
    - Spring Data JPA
    - RESTful API
    - Open API Documentation
    - PostgreSQL Database
    - JUnit 5
<br>

- **Frontend**
    - Angular 16
